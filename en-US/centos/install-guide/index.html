Installation Guide

Installing CentOS 7.6.1810 on all architectures

This manual explains how to boot the CentOS 7.6.1810 installation program (Anaconda) and how to install CentOS 7.6.1810 on AMD64 and Intel 64 systems, 64-bit ARM systems, and 64-bit IBM Power Systems servers. It also covers advanced installation methods such as Kickstart installations, PXE installations, and installations over VNC. Finally, it describes common post-installation tasks and explains how to troubleshoot installation problems.

Information on installing CentOS Atomic Host can be found in the Red Hat Enterprise Linux Atomic Host Installation and Configuration Guide.
